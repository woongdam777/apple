img 태그

        - 웹 문서에서 사진이나 그림 같은 이미지를 삽입하는 용도의 태그

        [속성]
        
        1. src : 삽입할 이미지의 경로를 작성하는 속성(파일경로, 웹주소)
        2. width / height : 이미지 크기(px, %)
        3. alt (대안): 이미지의 설명을 작성하는 속성
            -> 이미지가 출력되지 않는 경우 화면엑 표시됨
                +웹 접근성(스크린 리더가 이미지 설명을 읽어줌)

고정 크기 단위(px) - 항상 지정된 크기를 지님

가변 크기 단위(%) - 화면 또는 영역에 대한 상대적인 크기단위
